About ARCMF

Features

ARCMF is a great entry point for novice mission makers. Without a framework, making a mission can be a daunting task; gear scripts, weather scripts and all other sort of things have to be scripted, tested, and verified. With ARCMF, we've done that for you. Now, you can fully focus on your mission idea and bring it to life. No hassle, no fluff.

Development

Currently, the original ARCMF Team is developing ARCMF2, a continuation of ARCMF. A lot of new features await, such as 3DEN Editor integration and much more. ARCMF2 will be all the more powerful, and all the more accessible to the newcomer. We already think it's great, and we aren't even done yet!

Mission makers with a bit more experience will also find a lot of features to enjoy with ARCMF. Everything can be customised. Want to change automatic fireteam colors? Go do it. Want to change squad marker names? Go do it. Don't want to change anything? The don't, ARCMF has already done it for you.

With ARCMF2, we are also providing a whole new set of documentation once it is feature complete. You can always follow development over at the Github Repository, or check out the new website which is coming soon™. A list of what's new will be posted on the website as well as the readme file (once we get to it!).
